OPEN SOURCE HARDWARE

OpenAQ-Transit

A Low-Cost Curb-to-Cabin PM Monitoring Sensor Network for Sub-Saharan Africa.

55production units
deployed in Nairobi & Kigali

02 — System Architecture

The network integrates twin optical PM sensors on staggered duty cycles for in-field drift tracking.

  • Bus-Stop Units: 15W solar-powered nodes.
  • Bus-Mounted Units: Drawing power from vehicle rails with PIR-triggered cameras for source attribution.

03 — Deployment & Reliability

Data is uploaded via cellular link supporting over-the-air firmware updates. Calibration achieved < 15% bias in co-location tests.
